Python

A high-level, interpreted language known for its readability and large standard library.

    1. Variables & Basic Types

    Python is dynamically typed. Variables are created upon assignment. Basic types are numbers, strings, and booleans.

    
# No type declaration needed
name = "SyntaxForge"  # String
version = 3.12        # Float
year = 2024           # Integer
is_cool = True        # Boolean

# f-strings for formatted output (Python 3.6+)
print(f"{name} is cool: {is_cool}")

# Constants (by convention, not enforced)
PI = 3.14159

    2. Data Structures

    Python's core data structures are lists, tuples, dictionaries, and sets.

    
# List (mutable, ordered collection)
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(f"First fruit: {fruits[0]}")

# Tuple (immutable, ordered collection)
point = (10, 20)
print(f"X coordinate: {point[0]}")

# Dictionary (mutable, unordered key-value pairs)
person = {
    "name": "Alice",
    "age": 30
}
person["city"] = "New York"
print(f"{person['name']} is {person['age']} years old.")

# Set (mutable, unordered collection of unique elements)
unique_numbers = {1, 2, 3, 2, 1}
unique_numbers.add(4)
print(f"Unique numbers: {unique_numbers}")

    3. Control Flow

    Control flow is managed with `if/elif/else`, `for` loops, and `while` loops. Indentation is significant.

    
# If, Elif, Else
age = 18
if age < 13:
    print("Child")
elif age < 20:
    print("Teenager")
else:
    print("Adult")

# For loop (iterating over a list)
for fruit in ["apple", "banana"]:
    print(fruit)
    
# For loop with range
for i in range(3):
    print(f"Loop {i}")
    
# While loop
count = 0
while count < 3:
    print(f"Count is {count}")
    count += 1

    4. Functions

    Functions are defined with the `def` keyword. They can accept arguments and return values.

    
# Simple function
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))

# Function with default arguments
def power(base, exponent=2):
    return base ** exponent

print(f"3 squared is {power(3)}")
print(f"2 cubed is {power(2, 3)}")

# Function with arbitrary arguments
def add(*numbers):
    total = 0
    for num in numbers:
        total += num
    return total
    
print(f"Sum: {add(1, 2, 3, 4)}")

    5. Object-Oriented Programming

    Python is an object-oriented language. Classes are defined with the `class` keyword.

    
class Dog:
    # Class attribute
    species = "Canis familiaris"
    
    # Initializer / Instance attributes
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    # Instance method
    def bark(self):
        return "Woof!"

# Create an instance of the Dog class
my_dog = Dog("Buddy", 3)

print(f"{my_dog.name} is a {my_dog.species} and says {my_dog.bark()}")

    6. Error Handling

    Exceptions are handled using `try`, `except`, `else`, and `finally` blocks.

    
try:
    result = 10 / 0
except ZeroDivisionError:
    print("You can't divide by zero!")
except TypeError:
    print("Wrong data type!")
else:
    # This runs if no exception occurred
    print(f"Result was {result}")
finally:
    # This always runs, regardless of exceptions
    print("Execution finished.")

    7. Comprehensions

    A concise, Pythonic way to create lists, dictionaries, and sets.

    
# List comprehension
squares = [x**2 for x in range(5)]
# Result: [0, 1, 4, 9, 16]

# Dictionary comprehension
square_map = {x: x**2 for x in range(5)}
# Result: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# Set comprehension
unique_squares = {x**2 for x in [1, -1, 2, -2]}
# Result: {1, 4}

# Generator expression (memory efficient)
squares_gen = (x**2 for x in range(1000))

    8. File I/O

    The `with` statement is the recommended way to work with files as it handles closing them automatically.

    
# Writing to a file
with open("example.txt", "w") as f:
    f.write("Hello from SyntaxForge!\n")
    f.write("This is a new line.\n")

# Reading from a file
with open("example.txt", "r") as f:
    content = f.read()
    print(content)

    9. Common Standard Modules

    Python's "batteries-included" philosophy means it has a rich standard library.

    
import os
import datetime
import json
import random

# os module: Interact with the operating system
current_dir = os.getcwd()
print(f"Current directory: {current_dir}")

# datetime module: Work with dates and times
now = datetime.datetime.now()
print(f"Current time: {now.strftime('%Y-%m-%d %H:%M:%S')}")

# json module: Encode and decode JSON
data = {"name": "Forge", "version": 1}
json_string = json.dumps(data)
print(f"JSON string: {json_string}")

# random module: Generate random numbers
random_num = random.randint(1, 10)
print(f"Random number: {random_num}")

    1. Simple Web Scraper

    A script to fetch and parse the title of a webpage using `requests` and `BeautifulSoup`.

    
# Install libraries: pip install requests beautifulsoup4
import requests
from bs4 import BeautifulSoup

def get_page_title(url):
    try:
        response = requests.get(url, timeout=5)
        response.raise_for_status()  # Raise an exception for bad status codes
        
        soup = BeautifulSoup(response.text, 'html.parser')
        title = soup.find('title').text
        return title
    except requests.exceptions.RequestException as e:
        return f"Error fetching URL: {e}"

if __name__ == "__main__":
    title = get_page_title("https://www.python.org/")
    print(f"The title of the page is: '{title}'")
